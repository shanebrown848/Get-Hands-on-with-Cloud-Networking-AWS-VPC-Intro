## Overview

This project introduces core cloud networking concepts using Amazon Web Services. The focus is on understanding how cloud networks work and how AWS Virtual Private Cloud (VPC) is used to create secure, isolated network environments in the cloud.

Before cloud networking, organizations relied on physical networking hardware that required manual setup, maintenance, and scaling. Cloud networking replaces this with virtualized, on-demand resources that are faster to deploy and easier to manage.

---

## What I built

I created the foundation of a cloud network using **Amazon VPC**, which allows AWS resources to run inside a private, logically isolated network. This setup forms the base for future networking components like subnets, routing, and security controls.
